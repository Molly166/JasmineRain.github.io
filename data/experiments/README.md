# 实验数据存储目录

## 📁 目录说明

此目录用于存储实验记录和实验数据文件。

## 📂 目录结构

```
data/experiments/
├── 算法实验/
│   ├── 排序算法/
│   └── 图算法/
├── 数据库实验/
├── 机器学习实验/
└── ...
```

## 📝 使用说明

1. 将实验数据文件放在对应的实验文件夹中
2. 在文章或实验记录页面中使用相对路径链接文件
3. 文件会随博客一起部署，可以通过链接直接下载

## 🔗 链接示例

```markdown
[下载实验数据](./data/experiments/实验1/data.csv)
```

## ⚠️ 注意事项

- 单个文件建议小于 50MB
- 大文件建议压缩或使用外部存储
- 注意保护敏感数据
